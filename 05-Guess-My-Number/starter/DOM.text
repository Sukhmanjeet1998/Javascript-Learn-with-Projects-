Document Object Model: Structured presentation of HTML documents. Alllows Javascript to access HTML elements and styles to manipulate them.

document.querySelector(); To select elements
document.querySelector().textContent; To select elements context

DOM methods and properties for DOM manipulations are not part of JS, but they are WebAPI's made in JS and help in interacting with JS and help's in manipulating HTML elements.
API: Application Programming Interface

Event Listiner is something that happens on the page : Mouse click, Mouse movement, Key Press etc...
